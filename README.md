# Convex Hull Visualizer

Simple program to visualize the convex hull of a set of 2d points.

Uses [Grahams Scan](http://en.wikipedia.org/wiki/Graham_scan)

##Dependencies
* QT
  * Available from [homebrew](http://brew.sh/): brew install qt

## Build
1. qmake
2. make   

## Run
1. ./run
